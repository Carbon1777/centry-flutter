# ТЗ: Переработка приватного чата до уровня внутрипланового

## Цель

Привести приватный чат 1-на-1 к тому же уровню качества, функциональности и визуального стиля, что и внутриплановый чат. Сейчас приватный чат — это минималистичная реализация «текст + отправить». Нужно сделать его премиальным и функциональным.

## Текущее состояние

### Внутриплановый чат (эталон)
- **Файлы**:
  - `lib/ui/plans/widgets/plan_chat_sheet.dart` (~2000 строк) — основной UI
  - `lib/ui/plans/widgets/plan_chat_message_bubble.dart` — пузыри сообщений
  - `lib/ui/plans/widgets/plan_chat_avatar.dart` — аватары
  - `lib/ui/plans/widgets/plan_chat_block.dart` — блок чата

### Приватный чат (нужно доработать)
- **Файлы**:
  - `lib/ui/private_chats/private_chat_screen.dart` (~695 строк)
  - `lib/ui/private_chats/private_chats_list_screen.dart` — список чатов

## Что нужно перенести / реализовать

### 1. Визуал пузырей сообщений

**Сейчас в приватном чате:**
- Мои: `colors.primary` с alpha 0.85, без бордера
- Чужие: `colors.surface`, без бордера
- Радиус: 16pt с асимметричным уголком 4pt
- Нет аватара в сообщении
- Нет никнейма
- Нет даты/времени

**Нужно как в плановом:**
- Мои: `Color(0xFF10233F)` с бордером `Color(0xFF2A62C7)` @ 0.50
- Чужие: `theme.colorScheme.surface` @ 0.94 с бордером белого @ 0.14
- Удалённые: `surface` @ 0.40, текст «Сообщение удалено» курсивом
- Радиус: `BorderRadius.circular(18)`
- Padding: 12pt со всех сторон
- Бордер: 1pt
- Аватар 34pt слева от пузыря (для чужих сообщений)
- Никнейм над текстом (мои — `Color(0xFF7FB0FF)`, чужие — `Color(0xFF8BE4D4)`, fontWeight 800)
- Дата/время: формат `dd/MM HH:mm`, белый @ 0.68, fontWeight 500

### 2. Цитирование (reply)

**Сейчас:** отсутствует

**Нужно:**
- Свайп вправо по сообщению → режим цитирования
- Баннер над полем ввода с текстом цитаты
- Формат в пузыре: блок `_QuoteBlock` внутри сообщения
  - Фон: `Colors.black` @ 0.25 (мои) / 0.15 (чужие)
  - Левый бордер: 3pt синий `Color(0xFF3B82F6)`
  - Никнейм: italic, white @ 0.50
  - Текст: white @ 0.55, maxLines: 2

### 3. Редактирование сообщений

**Сейчас:** отсутствует

**Нужно:**
- Долгий тап → меню → «Редактировать» (только свои)
- Баннер «Редактирование сообщения» над полем ввода (аналог `_PlanChatEditBanner`)
- Отметка «Изменено» на отредактированных сообщениях
- Серверная поддержка: добавить `edited_at` в `private_chat_messages`

### 4. Удаление сообщений

**Сейчас:** только удаление чата целиком

**Нужно:**
- Долгий тап → меню действий (`_MessageActionSheet`)
- «Удалить у себя» — сообщение скрывается только у текущего пользователя
- «Удалить у всех» — tombstone (текст заменяется на «Сообщение удалено»)
- Режим выбора (selection mode) — checkboxes для массового удаления
- Серверная поддержка: RPC для удаления сообщений приватного чата

### 5. Emoji picker

**Сейчас:** отсутствует

**Нужно:**
- Кнопка emoji слева от поля ввода (toggle keyboard ↔ emoji)
- Встроенный пикер с категориями (аналог `_ChatEmojiPicker`)
- 8 категорий, ~700 эмодзи

### 6. Composer (поле ввода)

**Сейчас:**
- TextField: hint «Сообщение…», borderRadius 20pt
- Кнопка отправки: `IconButton` с иконкой
- Нет кнопки emoji

**Нужно:**
- Emoji button слева: `emoji_emotions_outlined` / `keyboard_alt_outlined`
- TextField:
  - Hint: «Напишите сообщение…»
  - FillColor: `theme.colorScheme.surface` @ 0.90
  - ContentPadding: 14/12
  - BorderRadius: 18pt
  - FocusedBorder: `Color(0xFF3B82F6)`
- Send button: `FilledButton` 48x48, borderRadius 16pt, иконка `arrow_upward_rounded`
- Фон контейнера: `Colors.black` @ 0.08

### 7. Меню действий при долгом тапе

**Сейчас:** отсутствует

**Нужно:**
- Долгий тап по сообщению → bottom sheet с опциями:
  - «Ответить» (→ цитирование)
  - «Редактировать» (только свои)
  - «Удалить у себя»
  - «Удалить у всех» (только свои)
  - «Скопировать текст»

### 8. Разделитель непрочитанных

**Сейчас:** отсутствует

**Нужно:**
- Розовая дашированная линия с текстом «Новые сообщения» при наличии непрочитанных
- Авто-скролл к этому разделителю при открытии чата

## Серверные изменения

1. **Таблица `private_chat_messages`**: добавить колонку `edited_at timestamptz`
2. **RPC `edit_private_chat_message_v1`**: редактирование текста сообщения (проверка авторства)
3. **RPC `delete_private_chat_message_for_me_v1`**: пометить сообщение удалённым для текущего пользователя
4. **RPC `delete_private_chat_message_for_all_v1`**: tombstone (проверка авторства)
5. **RPC `delete_private_chat_messages_bulk_v1`**: массовое удаление

## Что НЕ менять

- Навигация (AppBar с кнопкой назад — остаётся, это не sheet как в планах)
- Шапка чата (аватар + никнейм + имя — текущий дизайн ок)
- Меню «Удалить чат» / «Удалить и заблокировать» — остаётся как есть
- Механика realtime / polling — работает, не трогать
- Список приватных чатов — не входит в эту задачу

## Принцип работы

1. **Исследовать** все виджеты внутрипланового чата
2. **Вынести общую основу** (пузыри, аватары, emoji picker, action sheet) в переиспользуемые виджеты или скопировать с адаптацией
3. **Реализовать** серверные RPC для новых операций
4. **Интегрировать** в приватный чат
5. **Не сломать** внутриплановый чат
