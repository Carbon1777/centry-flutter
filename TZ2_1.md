## Шаг 2: Python-скрипт `scripts/select_best_photo.py`

### 2.1 Детекция лиц (КРИТИЧНО)

Для каждого фото:
1. Загрузить через OpenCV
2. Применить Haar cascade (`haarcascade_frontalface_default.xml`)
3. Определить размер обнаруженных лиц относительно размера изображения
4. **Крупное лицо** = ширина лица > 8% ширины изображения → фото НЕ подходит
5. **Мелкие фигуры** (< 5% кадра) — допустимо (фоновые прохожие)

### 2.2 Скоринг фото (из чего выбирать)

Для каждого фото без крупных лиц рассчитать score:

| Фактор | Вес | Логика |
|--------|-----|--------|
| Размер файла | +3 | Больше = больше деталей = скорее интерьер |
| Тёплые тона | +2 | Средний hue 10-40° (тёплое освещение = интерьер) |
| Яркость средняя | +1 | Средняя яркость 80-180 (не пересвет, не темнота) |
| Насыщенность | +1 | Высокая насыщенность (> 50) = яркий интерьер |
| Таймстамп в углу | -5 | Текст-паттерн в нижнем правом углу (дата съёмки) |
| Порядковый номер | +0.5 для _2 | Яндекс часто ставит интерьер вторым фото |

### 2.3 Приоритеты выбора

1. **Интерьер заведения** — зал, барная стойка, столики (лучший вариант)
2. **Фасад** — приемлемо если интерьера нет
3. **Еда/напитки** — приемлемо для ресторанов/кафе
4. **НЕ использовать**: фото с крупными лицами людей

### 2.4 Алгоритм

```
для каждого места:
  загрузить все фото (1-3 шт)
  для каждого фото:
    проверить лица → если крупное лицо → пометить face_detected
    рассчитать score
  отсортировать по score DESC
  выбрать лучшее без крупных лиц
  если все фото с лицами → пометить photo_needs_placeholder = TRUE
```

## Шаг 3: Обработка edge cases

- **Все фото с лицами** → `photo_needs_placeholder = TRUE`, `notes = 'all_photos_have_faces'`
- **Одно фото, есть лицо** → placeholder
- **Фото битое / не читается** → пропустить, пометить в notes
- **Таймстамп** — определить по наличию цифрового паттерна `20\d{2}` в нижней 15% области кадра

## Шаг 4: Обновление staging_places_normalized

```sql
UPDATE staging_places_normalized
SET selected_photo_index = {best_index},
    selected_photo_path = 'scraped_photos/{place_id}' || CASE WHEN {best_index} = 1 THEN '.webp' ELSE '_{best_index}.webp' END,
    photo_needs_placeholder = {has_faces_only},
    notes = {detection_notes}
WHERE place_id = '{place_id}';
```
