# TZ9 — Верификация, QA, перезапуск Cold Start

> Сессия 4 из 4. Зависимости: TZ6 + TZ7 + TZ8 завершены

## Цель сессии

Убедиться что всё работает в клиенте, перезапустить cold start с новыми правилами подбора мест.

---

## Шаг 1: Проверка RPC и клиента

### 1.1 RPC-функции
Проверить что все возвращают корректные данные:
- `get_places_feed_v5` — лента мест
- `get_feed_nearby` — ближайшие места
- `get_places_map_v2` — карта мест
- `get_place_details_meta_v1` — детали места
- `get_place_details_v2` — полные детали

### 1.2 Фото
- photos[].storage_key → brand-media bucket → getPublicUrl → изображение отображается
- is_placeholder = true → категорийный плейсхолдер отображается
- BoxFit.cover в клиенте

### 1.3 Контакты
- Телефон кликабелен (url_launcher, tel:+7...)
- Сайт кликабелен (url_launcher, https://...)
- Формат телефона: +7 (XXX) XXX-XX-XX в UI

### 1.4 Рейтинг
- Отображается корректно (★4.8, ★5.0)
- Голосование лайк/дизлайк работает
- Рейтинг пересчитывается

## Шаг 2: Контрольные проверки

### 2.1 Качество данных
- [ ] Нет мест-мусора (магазины, школы, доставки, киберспорт)
- [ ] Все 8 городов покрыты
- [ ] Адреса чистые — без этажей, подъездов, описаний маршрута
- [ ] Сайты открываются с https://
- [ ] Телефоны в формате +7XXXXXXXXXX
- [ ] Фото не содержат крупных лиц
- [ ] Рейтинг в диапазоне [1, 5]
- [ ] Все места имеют area_id
- [ ] Все места имеют координаты
- [ ] UNIQUE INDEX работает — дублей нет

### 2.2 Статистика
Вывести финальный отчёт:
- Мест по городам и типам
- Покрытие: фото, телефоны, сайты, рейтинг, метро
- Сравнение со старой базой (6110 → ~11000)

---

## Шаг 3: Перезапуск Cold Start

### 3.1 Новые правила подбора мест
- rating ≥ 4.5
- review_count — максимальный (из enrichment)
- Без забегаловок с 5.0 и 10 отзывами
- Приоритет: review_count * rating (взвешенный скор)

### 3.2 Генерация планов
- Фейковые юзеры получают планы вокруг проверенных мест
- Каждый город покрыт
- Разнообразие типов (не только бары)

### 3.3 Верификация cold start
- Планы отображаются в ленте
- Места в планах качественные
- Фото мест в планах корректные
